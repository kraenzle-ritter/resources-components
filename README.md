## Installation

Via Composer

``` bash
$ composer require kraenzle-ritter/gnd-lw-component
```

## Usage
Just put the component in your view:

```
@livewire('gnd-lw-component', ['params' => ['queryOptions' => ['size' => 5]]])
```
